# Wayfarer iOS App (SwiftUI)

Native iOS companion app for Wayfarer. Built with SwiftUI for fast iteration and a polished, iPhone-first UI.

## Requirements
- macOS + Xcode 15+ (SwiftUI)
- iOS 16+ target

## Run
1. Open `WayfarerApp.xcodeproj` in Xcode
2. Select an iPhone simulator (or your device)
3. Press **Run**

## Notes
- This repo is the native app shell with core screens and UI structure.
- API wiring will be added in the next iteration (chat, itinerary sync, map pins, etc.).

## Bundle ID
- Default: `com.wayfarer.app`
